# Pro hříbka
Pracuj v shellu.

**Smaž** si doku složku
a dej:

```sh
$ git clone https://github.com/kubp/daku.git
```
Kdyz udelas nejaky zmeny tak dej příkaz.
```
git status
```
Tady uvidis cos vsechno zmenil

Potom dej
```
git add *
git status
```
tim se všechni prida do commitu. Potom:

```
git commit
git push origin master
```
